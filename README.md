# Guía de Minecraft para Principiantes
 
## Descripción
 
Este proyecto es una guía interactiva sobre Minecraft orientada a jugadores nuevos. Explica cómo sobrevivir los primeros momentos del juego, qué materiales recolectar, cómo craftear objetos básicos y a qué enemigos prestarle atención.

---
 
## Contenido
 
- Menú principal con navegación interna
- Sección de primeros pasos con lista de acciones ordenadas
- Tabla de materiales básicos del juego
- Guía de crafteo con objetos esenciales
- Información sobre mobs y enemigos
- Video tutorial embebido de YouTube
- Formulario de contacto

## Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Variables CSS, Media Queries)
- Google Fonts (Press Start 2P)
- YouTube Embed (iframe responsivo)
- Git y GitHub Pages

---

## Mejoras visuales incorporadas

- Paleta de colores oscura temática de Minecraft con variables CSS
- Tipografía pixelada con Press Start 2P para títulos
- Tarjetas con efecto hover y sombra animada
- Navegación con transición de color en hover
- Layout con CSS Grid en la sección de Mobs
- Video responsivo que se adapta a cualquier pantalla
- Diseño responsive con media queries para tablet y celular
- Fondo con imagen y overlay semitransparente
- Botón de envío con animación de escala al hover
